About

This website shows some of the back-end abilities of Marga who owns and developed it by her own as part of the Full Stack Web Developer Course which took place in English language. It was designed for computers with recently updated browsers and the columns had been deisgned for no meaningfull display for mobile phones or smaller tablets, even if the content is displayed.

The website resides on a popular webserver and had been equiped with highest security measures. It can be remotely accessed only by Marga on the basis of the special port number chosen, her private key and additional passwords. Relevant files and directories had been secured (chmod), other accounts had been disabled.

Furthermore, this website consists of a CRUD (create,read,update, delete) functionality, routing, uses sessions state to keep information about the login status of the user. Moreover, it has a JSON endpoint, a secured deleting option ('delete' button has been secured so that only logged in users can delete items that they'd created before). It had been built as a mod WSGI app in Python flask with Postgre SQL database. Last but not least, the front end appearance had been built in bootstrap framework.

Please be informed that the 3rd party (Google) authorization signin/out (authorization token/client id, RESTfull API, AJAX call etc.) did correctly function initially. Currently, the login button does not enable a correct login and proper logout, because the Google's limit for developer account for 100 logins had been exceeded and therefore Google dismisses logins from this domane now.

This project had been built with iterative development and all code is pep,html,css compliant and of course neetly formatted.

This site is for testing reasons only and all information about the users are instantly deleted after their logout. Please feel free to add a new accounting item which you can edit afterwards.
